# ANTIGRAVITY - Tabi Planner

このドキュメントは、旅行計画アプリ「Tabi Planner」における**最上位の全体設計およびルール**を定義するものです。

## 1. プロジェクトの目的とコアバリュー

- **目的**: 家族間で共有できる、スマホ対応の旅行計画書（しおり）アプリを、コストゼロで構築する。
- **コアバリュー**:
  - **スマホ最優先**: 旅先でスマホから素早く確認・編集できるUI
  - **コストゼロ**: GAS + Google Sheets + HTML の完全無料構成
  - **家族共有**: スプレッドシートを共有するだけで全員が閲覧・編集可能
  - **前後の電車が一目でわかる**: 遅延時の代替便を事前に登録しておける設計

## 2. システム構成

- **データベース**: Google Spreadsheet（旅行一覧・スケジュール・交通手段の3シート）
- **バックエンド**: Google Apps Script (GAS)
- **フロントエンド**: HTML/CSS/JS（GAS Web Apps として配信）
- **地図**: Google Maps Embed API（無料・無制限、transit経路表示）

## 3. スプレッドシートのシート構成

| E | 終了時刻 |

| H | 場所 |

| 列 | 内容 |
|----|------|

## 5. 運用ルール（AIの行動指針）

- **破壊的変更の禁止**: スキーマ変更を行う場合は必ずユーザーへ提示し承認を得ること
- **セキュリティ**: Google Maps APIキーはスクリプトプロパティに保存し、コードに直書きしないこと
- **コストゼロ方針**: 有料APIは使用しないこと（Maps Embed APIは無料・無制限）
